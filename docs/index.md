## How to use this site

**During a project** — use the search bar (top right) or the left navigation to find exactly what you need. Most tutorials are written to be referenced mid-task, not read start to finish.

**Before class** — lecture slides are posted before or shortly after each session. The site is searchable, so if you remember a keyword from lecture you can usually find the relevant page fast.

**Something's missing or broken?** Email me or post in Webcourses. If a page has a mistake, the edit history is visible and I'll note corrections at the top of the page.

---

!!! tip "First time here?"
    Start with the navigation tabs at the top. Each tab maps to one of the three content areas above. The search bar works across everything — it's the fastest way to find a specific function, concept, or assignment step.
